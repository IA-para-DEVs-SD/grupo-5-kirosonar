# Prompts Utilizados

## Prompt para Atualização do README

O seguinte prompt foi utilizado para gerar o README do projeto conforme o padrão definido pelo professor:

```
conforme o padrao de projeto estabelicido pelo professor atualize o readme para o padrao
```

Arquivo de referência utilizado como contexto: `backend/docs/padrao-projeto.md`

### Padrão de tópicos exigido

1. Nome do Projeto
2. Breve descrição do projeto
3. Sumário de documentações
4. Tecnologias utilizadas
5. Instruções de instalação / uso
6. Integrantes do grupo
